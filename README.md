recuerda mantener las apis seguras bien para ejecutar el bot necesitas una terminal de linux, si eres de android utilizaras termux para instalarlo es cencillo tendras que ir a [F-droid](https://f-droid.org/es/) y descargar la app luego de que sete instale esperas, cuando ya este listo pones termux en el buscador te aparecera como termux gestor de paquetes tiene un logo asi
  
  
  <img src="https://upload.wikimedia.org/wikipedia/commons/b/b5/Termux.svg" alt="Termux logo" width="100">


asi se ve el logo de la app de termux despues cuando sete descargue termux pones 
```bash
pkg upgrade $$ pkg update
```
si te aparece algo como [N/Y] pones y que es yes y n es no, y si ya lo pusistes le das a return que es enter
luego de eso pondras instalar el proyecto que es con
```bash
git clone https://github.com/garenafreefire-max/Avestrus-bot/
```
esto es para instalar el proyecto, ya que tengas instalado el proyecto osea el bot pondras
```bash
pkg install ffmpeg nodejs python python3 python-pip librsvg git yarn yt-dlp unzip -y
```
luego de que sete alla instalado eso pones
```basg
unzip avestrus-wa
```
luego de que sete alla descomprimido pondras
```bash
cd avestrus-wa
```
esto es para entrar ala carpeta el cd, y de ultimo pondras
```bash
npm install
```

si estas en sdcard o storage/emulated/0 es lo mismo pero tendras que usar 
```bash
npm install --no-bin-links
```
